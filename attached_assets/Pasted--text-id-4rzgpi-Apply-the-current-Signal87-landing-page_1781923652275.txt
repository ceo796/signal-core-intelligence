Use the same or closest available landing page font system.
The dashboard should feel like the landing page:
- large, elegant headings
- refined spacing
- warm off-white text
- muted gray secondary text

3. Accent color
Use the landing page green/verification palette:
- dark green / emerald accents
- subtle mint-green highlights
- use green for “verified,” “connected,” “ready,” and citation/source trust states
- avoid bright blue/purple as the primary accent on this page

4. Cards and panels
Cards should feel like the floating interface panels on the landing page:
- dark glass-like panels
- thin muted borders
- subtle shadows/glow
- off-white document preview cards where useful
- rounded corners
- layered depth

5. Documents area
Recent documents and the Documents page should feel premium:
- show real documents only
- thumbnails/previews where available

- no chunks
- compact metadata
- clear actions
- hover states with subtle lift/depth

6. Ask / AI entry area
The main Ask Signal87 bar should feel central and premium:
- dark panel
- soft green focus glow
- warm off-white text
- source/verification language
- route to existing AI Chat / Ask page
- do not create a new chat backend

7. Motion
Add restrained premium interaction:
- subtle card lift on hover
- pointer-responsive depth only where safe
- smooth transitions
- soft section reveal
- hover movement on arrows/buttons
- respect prefers-reduced-motion
- no heavy WebGL
- no gimmicky animation

- real recent documents if available

- Home / Command Center
- large ask bar
