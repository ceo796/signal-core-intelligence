Implement the new simplified Signal87 AI dashboard design based on the latest approved mockup.

Core design decision:
Use the simplified layout where “Recent documents” and “Recent briefs” are merged into one unified section called “Recent work.” Do not keep them as separate cards. Remove the “Intelligence at a glance” panel entirely.

Strict scope:
This is a visual and layout refinement only. Do not change backend logic, authentication, routing, document upload logic, chat logic, brief generation, APIs, database queries, environment variables, Clerk/Supabase/OpenAI configuration, or protected platform functionality.

Design direction:
Create two theme-ready versions of the same layout:

1. Cream/light version:

   * Cream or warm off-white background
   * Black/deep ink typography
   * Subtle warm borders
   * Very restrained navy/black accents
   * No purple as the dominant color

2. Dark version:

   * Black or near-black background
   * Cream and white typography
   * Subtle graphite panels
   * Warm cream/gold accent for active states and primary buttons
   * No bright purple as the dominant color

Typography:
Use an agentic, technical sans-serif system. Do not use serif fonts.
Preferred:

* Geist Sans
* IBM Plex Sans
* Inter Tight
* Space Grotesk only if it remains professional

Avoid:

* Serif headings
* Playful rounded typography
* Generic SaaS font treatment
* Editorial or consulting-style typography

Required layout:

* Left sidebar navigation remains.
* Top area includes welcome headline and AI command/search bar.
* Quick action row remains, but should feel compact and calm.
* Main content area includes:

  * One large “Recent work” section combining documents and briefs.
  * One large “Signal87 AI” panel below it.
* Right column includes:

  * Recent activity
  * Suggested actions
* Remove:

  * Separate “Recent documents” card
  * Separate “Recent briefs” card
  * “Intelligence at a glance” card
  * Any analytics/sparkline/metrics panel

Recent work section:
Show mixed items in one list with a small type label/pill such as:

* Document
* Brief

Example items:

* Q2 Market Research Report.pdf — Document — Uploaded 2h ago
* Q2 Growth Opportunities Brief — Brief — Generated 2h ago
* Competitive Landscape.xlsx — Document — Uploaded 1d ago
* Competitive Analysis Brief — Brief — Generated 1d ago
* Investor Update — May 2025.pdf — Document — Uploaded 2d ago

The section should feel like a clean workspace activity table, not a card grid.

Visual rules:

* Reduce visual busyness.
* Use fewer panels.
* Reduce shadows.
* Use subtle borders.
* Reduce corner radius slightly.
* Avoid bubbly cards.
* Avoid bright gradients.
* Keep hierarchy focused on the AI command bar, Recent work, and Signal87 AI.
* Suggested actions should be helpful but visually secondary.
* Recent activity should be quiet and compact.

Sidebar:

* Keep minimal navigation.
* Active state should be subtle: thin vertical rail, muted background, or quiet highlight.
* Do not use bright purple pills.
* User profile remains at bottom.

Signal87 AI panel:
This should be the most important workspace element after Recent work.
Keep:

* Sources connected status
* Manage link
* Example question chip
* AI answer box
* Follow-up input
* Send button

Mobile:
Ensure the layout stacks cleanly:

1. Welcome
2. Command bar
3. Quick actions
4. Recent work
5. Signal87 AI
6. Recent activity
7. Suggested actions

No horizontal overflow. No cramped cards.

Implementation instructions:
First inspect the current frontend files, dashboard component structure, theme tokens, global CSS, Tailwind config, and existing styling system. Then make the smallest safe set of changes needed.

Prefer modifying:

* Dashboard layout component
* Dashboard-specific cards/sections
* Theme tokens or CSS variables
* Global font import/theme if necessary
* Tailwind theme values if already used

Avoid:

* Large rewrites
* New backend code
* New APIs
* New database calls
* New dependencies unless absolutely necessary for the font

Validation:
Run existing build/typecheck/lint commands if available. Verify:

* Dashboard renders correctly
* Navigation still works
* Upload document entry point still appears
* Create brief entry point still appears
* AI command/search input still appears
* Signal87 AI panel still appears
* Mobile layout stacks correctly
* No console errors
* No horizontal overflow
* No backend/auth/API files changed

Final report:
Provide:

* Files changed
* Components removed
* Sections merged
* Theme/color/font changes
* Checks run
* Confirmation that no backend, auth, API, database, or environment configuration was touched
