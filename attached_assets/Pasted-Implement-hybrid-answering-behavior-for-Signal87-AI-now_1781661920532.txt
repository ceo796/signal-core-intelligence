Goal:
Signal87 AI should answer simple/general questions even when no relevant uploaded document context exists, while still using grounded document answers with citations when the user asks about uploaded files.

Strict scope:
Do not change authentication, routing, document upload, brief generation, database schema, environment variables, Clerk/Supabase/OpenAI configuration, dashboard design system, file-type utility, sidebar layout, or unrelated platform functionality.

Required behavior:

1. General AI mode
   If the user asks a general question that does not require uploaded documents, answer normally without requiring citations.

Examples:

* What is a SAFE note?
* What does EBITDA mean?
* Draft a follow-up email.
* What is the difference between Seed and Series A?
* Explain this term.

For general answers:

* Do not fabricate citations.
* Do not imply the answer came from uploaded documents.
* Show a subtle label: “General answer — not grounded in uploaded documents.”

2. Document-grounded mode
   If the user asks about uploaded documents, selected documents, files, contracts, PDFs, summaries, comparisons, risks, clauses, citations, sources, or file contents, use the existing document retrieval and grounding system.

Examples:

* What does this document say?
* Summarize the uploaded contract.
* Compare these two PDFs.
* What are the risks in my uploaded file?
* Find the termination clause.
* What do my documents say about revenue?

For grounded answers:

* Preserve the existing citation/source behavior.
* Do not bypass retrieval when the user is clearly asking about uploaded files.
* If relevant sources exist, answer from those sources.

3. Hybrid mode
   If the question is ambiguous, provide a helpful general answer first, then clearly state whether document context was available.

Example:
“Generally, this means X. I do not see enough relevant document context to answer from your uploaded files. Upload or select a document and I can ground the answer with citations.”

For hybrid answers:

* Do not fabricate citations.
* Use a subtle label: “General answer. No relevant document source found.”
* If relevant document context exists, include a short “From your documents” section with citations.

Core trust rule:
The user must always be able to tell whether the answer came from:

* General AI knowledge
* Uploaded document context
* A hybrid fallback
